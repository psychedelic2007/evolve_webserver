EVOLVE Tutorial

Ever wondered if you can optimise the process of species evolution? Predict the future mutations in a species? With EVOLVE, it's totally possible!!!

Step -by- Step Guide

Click on individual Steps to know the details about how to perform the calculation

Step 1.

Downloading the Dataset

Follow this step to download the necessary dataset (protein and genomic sequences) from different databases.

Step 2.

Dataset Preprocessing

Follow this step to preprocess the raw data that you downloaded in step 1. To know about the theoretical background of how we are preprocessing the dataset, please go through the Theory Corner.

Step 3.

Entropy Calculation

Follow this step to calculate the entropy of the viral strain using its protein or genomic sequence. The entropy is calculated using the Shannon's entropy formula. To know more about the theoretical background of the entropy calculation please go through the Theory Corner.

Step 4.

Calculating MRF

Follow this step to calculate the Mutational Response Function (MRF) of the viral strain. MRF is new mathematical quantity developed by us. To know more about the MRF you can go through the following paper or please go through the Theory Corner.

Step 5.

Feature Calculation

Follow this step to calculate the Features for the model training. We have defined four features for the model training. These are: Pair Predictibility of Amino Acid (PPAA), Distribution Probability of Amino Acids (DPAA), Future Composition of Amino Acids (FCAA) and Entropy Calculation of Amino Acids (ECAA). To know more about the mathematical background of the features please go through the Theory Corner.

Step 6.

Model Prediction

Follow this step to train the Feedforward Machine Learning Model. You can select the appropriate hyperparameters for the model training. To know more about the hyperparameters of the model, please go through the Theory Corner.

Step 1: Downloading the Dataset

In this step, you will need to download a dataset of protein/genomic sequences either from the National Center for Biotechnology Information, Virus Database NCBI Virus or the Global Initiative on Sharing Avian Influenza Data GISAID. NCBI and GISAID are two major databases for accessing genetic sequence data, and you can choose either one based on the specific research question you are trying to answer.


To download the dataset, you will need to navigate to the NCBI Virus or GISAID website and search for the dataset you want to download. You can filter the search results based on the specific criteria you are interested in, such as the type of virus or the geographic region. Once you have found the dataset you want to download, you can download it in a compressed file format, such as .fasta or .fastq


In our previous work we downloaded the protein data from NCBI Virus, in a month-wise manner, by selecting certain filters. Follow the following steps to download the data in a month-wise manner

NCBI Virus Database to Download Protein Sequence

(A) Go to NCBI Virus dashboard


(B)Click on the Country for which you want to download the data. In this case we select India and then scroll up and select All Proteins


(C) Here you will find the details of all the viral proteins in that country. From the left panel select filters to narrow down the search to a specific variant. Click on Nucleotide Completeness and select Complete. This step makes sure that you are using a complete sequence and not a partial or incomplete sequence


(D) Once you have selected the complete sequence, click on Pango Lineage to select the variant of interest. In this case we are selecting the Delta Variant (B.1.617.2). Make sure that you enter the variant ID and not the generic name of the variant.


(E) Once you have selected the variant of your choice, then click on Proteins. Here you can select which viral protein you would like to investigate. In this case we have selected the Surface Glycoprotein of Delta Variant


(F) Click on Collection Date to display the sequences in a month wise manner


(G) Finally to download the sequences click on Download and proceed with the default options

GISAID Database to Download Genomic Sequence

You need to make an account in GISAID first. Go to GISAID and in the top right corner click on Log In. Create an account and once you log in you will see the home page


(A) Make sure you are on the EpiCov tab. Click on the Viral structure of SARS-CoV-2 and it will take you to the working dashboard


(B) Once you are on the working dashboard navigate to Location tab. Here you can search the genomic sequence based on the countries. But there is a format that you need to use to search. First type the name of the Continent followed by a space and then a Forward Slash, followed by the name of the country. For example if we need to search sequences from India then we will type Asia / India


(C) After searching for the country of choice, navigate to the right side to select three important features: Complete, High Coverage and Collection Date Complete. Selecting Complete assures that the sequences that you will be getting are not incomplete or broken. Selecting High Coverage affirms that the genomic sequences that will be displayed has less amount of unrecognised base pairs i.e. low value of N's (<1% N and <0.05% mutations). And finally selecting Collection Date Complete allows the server to give you entries with complete collection date.


(D) Once the filters are selected then you can navigate to Lineage tab to select the variant of your choice. For example if we want to download the genomic sequence of Delta variant then in the Lineage tab we will type the ID of the Delta variant (i.e. B.1.617.2).


(E) Once you have selected the variant of interest then you can navigate to the Collection Date and collect the sequences in a month wise manner


(F) After performing the above steps you will get the refined sequences in a month-wise manner. To download the sequences you can click on the Square Box on the left side of Virus Name tab. This will select all the sequences in the entry and then you can navigate to the bottom right and click on the Download button to download the dataset. Once you click on Download a new window will pop up asking you to select the format of the dataset. Click on Nucleotide Sequences (FASTA) to download the dataset in FASTA format and then click on Download. Then you will be asked to accept the terms and conditions of GISAID database. Click on I agree to the terms and conditions and then click Download.

Step 2: Dataset Preprocessing

(A) After downloading the dataset it is important to preprocess it as the raw data consists of many junk and broken data.To preprocess the data click on Preprocessing tab from the main menu and then upload the sequence file that you have downloaded in previous tep in FASTA format. Click on Submit after which you will get your preprocessed data. To know more about what exactly the server does in preprocessing please visit the Theory Corner.


(B) If you are unsure about which data file or what type of datafile you need to upload, you click on Load Example and then click on Submit. Once the preprocessing is done, you can download the file and then view it to understand the file type and its architecture.

Step 3: Entropy Calculation

(A) After you have preprocessed the data the first thing that we need to do is to calculate the entropy.


(B) Before you can calculate the Mutational entropy, you need to have the alignment file with you for the preprocessed data. You can visit MAFFT Server to perform the Multiple Sequence Alignment and Download the alignment file.


(C) On the MAFFT Server, upload your preprocessed FASTA file by clicking on "Choose File" and then click on "Submit". You can either keep the deafult values as it is or can change it according to your own need. Once the alignment is done, you can download the alignment file (.aln) from the top left corner by clicking on "Clustal Format".


(D) Upload the clustal alignment file (.aln) in our webserver and click on "Submit". You will get the desired output in the form of Mutational Entropy graph


(E) If you are unsure about which data file or what type of datafile you need to upload, you can click on Load Example and then click on Submit. Once the preprocessing is done, you can download the file and then view it to understand the file type and its architecture.


(F) In the output you will get the graph of Entropy vs Residue number. This graph depicts the Mutational Entropy of each residue in the viral strain

×

Step 4: Mutational Response Function

(A) After calculating the Entropy of the genomic/protein sequence you need to supply the data file in order to calculate the MRF


(B) When you download the entropy data in the previous step you will get a CSV file. If you look at the architecture of the CSV file you will see that the first column of the CSV file is the "Residue Position" and the second column is the "Entropy". You need to prepare the correct data in order to calculate the MRF as well


(C) The CSV file should have the following architecture: The first column will correspond to "Residue Position", The second column will correspond to "Entropy" of a particular month. Depending upon the average month that you are going to take (default is 3), you need to increase or decrease the number of months. So if you want to calculate the MRF for 5 months, for example, then the first column will be "Residue Position" and then second, third, fourth, fifth and sixth columns will correspond to the entropy of the individual months


(D) You can download the example file to see the architecture of the file.

Step 5: Feature Calculation

This step is totally different from the previous steps as in this step you will be calculating the features necessary to train the model. There are four major features that you are required to calculate in order to train the model


Feature 1: Pair Predictability of Amino Acids

(A) To calculate this feature click on "Feature 1" on the left panel in your dashboard


(B) Upload the preprocessed fasta sequence file and click on "Submit" and this will calculate the Feature 1 of all the sequences present in your fasta file


(C) In the output you will get a zip file to download. Download the zip file and extract it. The folder will consists of CSV files named based on the sequences's Accession ID


(D) The architecture of the CSV file will correspond to Residue Position in first column and Feature 1 for each amino acid in second column


To know more about the mathematical background of the Feature 1, please go through the Theory Corner

Phylogenetic Calculation

(A) Once all the features are calculated, you need to perform a phylogenetic analysis in order to get the target class labels


(B) To perform the phylogenetic analysis, you need to first get the phylogenetic tree. To get the phylogfenetic tree, upload the preprocessed fasta sequence file in MAFFT Server and then click on "Submit"


Once you get the output page, click on "Phylogenetic Tree" option


Once you click on "Phylogenetic Tree" option a new page will open up. Keep the parameters at their default values unless you want to change anything. Once you have decided on the parameters then click on "Go!"


Once you click on "Go!" a new window will come up. Click on "View Tree on Phylo.io" to view the phylogenetic tree


Then visit the Local Calculation directory of the official GitHub page of the webserver


Download the "phylogenetic_calculation.py" python file in your local computer and run the code


When prompted to enter the first sequence, then type the Accession ID of the root sequence (parent) and press enter. When prompted to enter the second sequence, then type the Accession ID of the branch sequence (daughter) and press enter. Make sure to change the "fasta_file_path" to a folder where you would like to save the files.


To know more about how to calculate the root sequence (parent) and branch sequence (daughter) please visit the Theory corner

Once you have calculated all the necessary features and performed the phylogenetic analysis again visit the Local Calculation directory of the official GitHub page of the webserver and download the python files starting with "combine" and then run them in your local computer to generate the combined files which will be the training dataset for the model


To know more about how to run the python codes, visit the "README.md" file in the "Local Calculation" directory

Step 6: Mutation Prediction

(A) After calculating the features for the preprocessed sequences you need to supply the data file for the mutation prediction


(B) To create a data file, create a CSV file consisting of only 4 columns: Feature 1, Feature 2, Feature 3, Feature 4 and Target


(C) Upload the file in the webserver by clicking on "Upload" button


(D) After uploading the file, click on submit and it will start the model prediction on your data.


After the prediction is done, you will see a prediction graph with residue position on X-axis and Mutational Probabilityon Y-axis. You can download the graph and the data in a csv format.
